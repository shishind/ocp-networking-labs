# OCP Networking Labs - Website

Docusaurus-powered website for the OCP Networking Labs curriculum.

## Quick Start

```bash
npm install
npm start
```

Opens at: http://localhost:3000

## Deploy to GitHub Pages

**Automatic deployment via GitHub Actions:**

1. Copy website to your repo:
   ```bash
   cp -r /root/claude/ocp-networking-website/. /root/claude/ocp-networking-labs/website/
   ```

2. Enable GitHub Pages:
   - Go to: Settings → Pages
   - Source: GitHub Actions

3. Push and it auto-deploys!

**Live URL:** https://shishind.github.io/ocp-networking-labs/

## Build

```bash
npm run build
npm run serve
```
